#### Steps for installing the project		
 Checkout this repo, install dependencies, then start the gulp process with the following:		
 		
 ```		
 > git clone https://github.com/MeenakshiParyani/Kayak.git		
 > cd front-end		
 > npm install		
 > npm install -g web-pack		
 > npm start
 
 > cd back-end
 > npm start
 
 > cd kafka-backend
 > npm start
